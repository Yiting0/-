// 工業相機做到什麼？
看得多精細？
解析度
感光元件上的像素愈多，影像能呈現的細節就愈多——就像手機從 800 萬升到 4800 萬畫素的差距。
移動中的物件拍得清楚嗎？
快門類型
一般相機拍移動中的東西會變形。工業相機有專門解決這個問題的技術——在選型時這是決定性的因素。
影像怎麼傳到電腦？
傳輸介面
不同的連接方式決定了能拍多快、相機離電腦能有多遠——兩者通常無法同時兼顧。
📐
解析度與 Pixel Size——精細度的兩個面向
像素愈多看得愈細，但像素愈大反而感光更好——要看場景決定
▼

解析度就是影像的「格子數」，格子愈多、細節愈豐富。但格子多不代表每格就一定看得清楚——每個格子（像素）的實體大小（Pixel Size）決定它能收集多少光。像素愈大，低光環境下雜訊愈低、影像愈乾淨；像素愈小，同樣面積能塞進更多格子，解析度更高，但在光線不足時影像容易出現雜訊。

選型時先確認最小可偵測的缺陷大小，確保那個缺陷至少能佔到影像上的 5 個像素以上，不然演算法根本「看不見」它。

💡
// 你知道嗎？
評估相機品質時要看兩個數字：SNR（訊號雜訊比）超過 39 dB 才算優質；DR（動態範圍）超過 58 dB 才能清晰呈現強烈明暗對比的場景。買之前記得確認規格表！
⚡
Rolling Shutter vs Global Shutter——物件在動就一定要選對
拍移動中的物件，選錯快門類型，邊緣會歪掉
▼

想像你用掃描機掃一張紙——掃描頭從上往下一行一行掃。如果掃到一半紙張突然移動，掃出來的影像就會錯位變形。Rolling Shutter 的原理和這個一樣，感光元件逐列曝光，物件只要在移動，邊緣就會歪斜——這在生產線上是很嚴重的問題。

ROLLING SHUTTER
逐列曝光
成本低。靜止或慢速物件適用。物件移動時邊緣出現果凍變形，本來的直線會變成斜的。
GLOBAL SHUTTER
全幅同步曝光
所有像素在同一瞬間同時曝光，物件怎麼動都不會變形。成本較高，是高速生產線的必要選擇。
🔌
傳輸介面——速度快，還是距離遠？
GigE 可以拉很遠，USB3 速度快好安裝，CoaXPress 兩個都要就選這個
▼

影像從相機傳到電腦就像水管送水——管子愈粗（頻寬愈大）每秒能傳的影像愈多，但管子拉得愈長阻力就愈大。

GigE Vision：走一般網路線，最遠能拉 100 公尺，適合機台大、相機離電腦很遠的場合，但頻寬只有 1 Gbps，高速場合可能不夠用。

USB3 Vision：就像電腦的 USB 插槽，佈線最簡單，速度也快（5 Gbps），缺點是距離只有 10 公尺，適合桌上型設備或實驗室快速建置。

CoaXPress：高階選項，頻寬最高達 25 Gbps，距離也可以到 68 公尺，連接線本身還整合了電源，不用另外拉電線。速度快、距離遠都要就選這個，但成本最高。

📡
線掃描相機——高速連續物料的最佳解
一次只拍一條線，連續掃過去拼成完整影像，適合捲材和連續帶狀物件
▼

一般面掃描相機（Area Camera）每次拍一張完整的矩形影像，適合靜態或低速物件。但如果要檢測連續移動的捲材——例如布料、薄膜、金屬板——面掃描相機就跟不上速度了。

線掃描相機（Line Camera）的感光元件只有一行像素，每次拍攝只擷取一條線，然後隨著物件移動不斷累積，最終拼合成一張完整的長條影像。只要物件速度夠穩定，就能得到完整無縫的檢測影像。

線掃描的三大優點
🔹 超高解析度：橫向像素數可達 8K 以上，細節遠超面掃描
🔹 連續不間斷：物件多長就能拍多長，無需分段
🔹 高速拍攝：線頻率可達數十 kHz，輕鬆跟上高速生產線

適合場景：印刷品質、紡織布料、薄膜表面、面板和金屬板的外觀檢查。需要搭配穩定的物件移動速度，且光源設計要夠亮（因為曝光時間極短）。

🎨
彩色相機的四種感光技術
Bayer 最便宜但解析度打折，3CCD 最好但最貴，各有取捨
▼

彩色相機不是把黑白影像染色——它的感光元件本身就設計成能分辨 RGB 三種顏色。但怎麼分辨，各家做法不同：

🔲 Bayer Pattern
✓ 成本最低、最普遍
✗ 解析度/銳利度下降（相鄰像素借色）
🔷 3CCD / 3CMOS
✓ 成像品質最佳（稜鏡分光至三片元件）
✗ 成本高、體積大、需精準對位
⬛ Pixel Shift
✓ 解析度佳，可做大尺寸
✗ 需移動元件，移動中物體會模糊
🟣 Foveon
✓ 三層感光，解析度無損
✗ 紅光感光差、夜拍效果不佳

工業上最常用 Bayer Pattern，夠用且成本低。需要頂級色彩品質時才考慮 3CCD/3CMOS。注意：彩色相機不適合用來拍灰階影像轉換，靈敏度不如黑白相機。

⚙️

三個旋鈕控制影像亮度，但各有代價，順序很重要
▼

F 5.6
工業建議範圍，畫質最穩定，景深適中。
F1.4（大）F22（小）

曝光時間（Exposure Time）：控制感測器感光的時間長短。時間愈長影像愈亮，但移動中的物體會模糊。工業用單位是 ms（毫秒）或 μs（微秒）。

增益（Gain / ISO）：光圈和曝光都調好後，用來補充亮度的最後手段。Gain 愈大影像愈亮，但雜訊也愈多——優先調前兩個，增益越小越好。

🏆 黃金原則：寧可稍暗，絕不過曝。過曝的亮部細節永遠救不回來；偏暗的影像後製還有機會。拍攝順序：先調光圈 → 再調曝光 → 最後才動增益。

// LENS SELECTION
鏡頭
決定「看多大、看多細」
鏡頭就像眼睛的焦距——拉遠看全景，拉近看細節，但兩件事沒辦法同時做到。選鏡頭的核心，就是在「視野大小」和「看得多細」之間找到剛好的平衡點。
// 鏡頭決定兩件事
一次能看多大的範圍？
視野（FOV）
倍率愈低，鏡頭「退得愈遠」，一次能框住的範圍愈大——就像站遠一點拍團體照。
最細能看到多小的東西？
像素解析度
影像裡每一個像素，代表現實中多大的一塊面積。這個數字愈小，就能看到愈細的缺陷。
⚠️ 兩者互相取捨——倍率調高看得更細但範圍縮小；想同時要求「又大又細」，只能換更大的感光元件。
🎯
Image Circle——鏡頭能投影多大的圓？
感光元件必須完整落在這個圓圈內，否則四角會黑掉
▼

鏡頭投影出來的清晰影像，形狀其實是一個圓，不是方形。相機裡的感光元件是矩形的，必須完整塞進這個圓裡面——如果感光元件的對角線比圓的直徑還長，四個角就會超出圓的範圍，那裡的影像會變暗、變模糊，這就是「暗角」。

選鏡頭時，確認感光元件的對角線長度小於鏡頭標示的 Image Circle 數值就沒問題。

🔬
遠心鏡頭——量測用鏡頭的最高規格
沒有透視失真，景深內倍率完全一致，精密量測的首選
▼

一般鏡頭有個無法避免的物理現象：近的東西看起來大、遠的東西看起來小。這在看照片時很正常，但在精密量測時就是致命缺陷——同一個零件放近一點和放遠一點，量出來的尺寸會不一樣。

遠心鏡頭（Telecentric Lens）解決了這個問題。它的光路設計讓只有平行於光軸的光線才能進入，使得景深範圍內的所有物件，不管距離遠近，倍率都完全一致。

一般鏡頭
有透視效應
物件靠近時看起來變大，離遠時變小。量測時必須確保工件放置高度完全一致，否則數字會跑掉。
遠心鏡頭
無透視失真
景深內倍率一致，工件就算有輕微的高低誤差，量測結果也不受影響。邊緣成像清晰穩定。

適合場景：半導體元件、精密五金零件、電子元件的尺寸量測。缺點是體積大且價格昂貴，FOV 也受限於鏡頭尺寸無法做太大。

🤔
// 你知道嗎？
遠心鏡頭的「Telecentric」字面意思是「遠處的中心」——光線必須平行進入，就好像光源在無限遠處一樣。這讓它的成本是一般鏡頭的 5～20 倍，但在精密量測場合無可替代。
🔩
C-Mount vs CS-Mount——差 5mm，搭錯就對不了焦
相機接口和鏡頭接口必須匹配，否則無法正確成像
▼

C-Mount 和 CS-Mount 是工業相機最常見的兩種鏡頭接口，外觀幾乎一樣，差別只在鏡頭裝上去之後，鏡片到感光元件的距離不同——C-Mount 是 17.526 mm，CS-Mount 是 12.5 mm，差了 5 mm。

這 5 mm 的差距如果沒有補上，鏡頭的焦點就不會落在感光元件上，整張影像都是模糊的。C-Mount 相機只能裝 C-Mount 鏡頭。CS-Mount 相機比較彈性，兩種都能用，但裝 C-Mount 鏡頭時，中間要夾一個 5 mm 的延伸環來補足。

📏
焦距、工作距離、光圈、畸變
選鏡頭前必須確認的四個實際限制
▼

焦距 f：焦距愈長，在相同倍率下工作距離就愈遠——這反而是好事。機台裡如果有障礙物擋住相機，選長焦距鏡頭可以把相機「退到更遠的地方」，繞開障礙還能保持同樣的拍攝倍率。

工作距離 WD：鏡頭前端到物件的距離。如果想讓鏡頭在更近的距離對焦，可以在相機和鏡頭之間夾入近攝接寫環（Extension Tube），不換鏡頭就能實現近距離拍攝。

光圈：光圈開大，進光量多、景深淺（只有一個平面清楚）；光圈縮小，景深加大（前後範圍都清楚），但影像變暗。檢測 3D 立體物件通常需要縮小光圈。

畸變：便宜的鏡頭會讓直線看起來微微彎曲。多數外觀檢測不在意這個，但如果要量測尺寸，就必須選低畸變的鏡頭，否則量出來的數字會有誤差。

// LIGHTING DESIGN
光源
讓缺陷自己浮現
光源設計是機器視覺裡最容易被忽略、卻最能左右結果的環節。同一個缺陷，打對光能讓它在黑暗中發亮；打錯光則讓它完全消失——後端再強的演算法都補救不了。
// 打光要做到兩件事
讓缺陷浮現
凸顯目標特徵
好的打光讓刮痕、汙點、缺口在影像裡變得明顯——就像用手電筒斜打牆壁，凹凸不平的地方全都現形。
讓背景消失
壓制背景干擾
不重要的背景紋理和反光如果太明顯，演算法會不知道該看哪裡。打光要讓背景盡量單純、乾淨。
💡 現代工業幾乎全用 LED——壽命長、亮度可精確控制，還能在快門瞬間閃出數倍亮度，凍結高速移動的物件。

LED 光源的形狀——不同形狀解決不同問題
環形光、條形光、背光板……形狀不同，打出來的效果完全不一樣
▼
💍 無影環光（Dome Light）
半球形漫射光源，從所有角度均勻打光，消除陰影和高反射面的眩光。適合金屬零件、電路板等高反光表面的外觀檢測。
⭕ 環形光（Ring Light）
圍繞鏡頭一圈的 LED 陣列，正面均勻照射。是最常見的通用型光源，適合一般表面外觀檢測。
📏 條形光（Bar Light）
單方向線性光源，強調特定方向的陰影與高度差。適合邊緣檢測、刮痕、焊錫高度量測，也常搭配線掃描相機使用。
🎯 外同軸光（Coaxial Light）
光線從鏡頭內部沿光軸方向射出，利用半反射鏡導光。適合極度平坦的高反光表面（如晶圓、金屬蓋板），能讓微細刮痕現形。
⬛ 背光板（Back Light）
放在物件背面，產生均勻亮面讓物件呈現黑色剪影。尺寸量測的首選——輪廓最清晰，量測誤差最小。
💡
各種光源的特色與選擇
鎢絲燈熱、螢光燈老化快、氙氣燈貴——現代工業幾乎全用 LED
▼

光源發展了很多種類，但每種都有它的取捨，最終 LED 幾乎成為工業視覺的唯一答案。

🔴 鎢絲燈 / 鹵素燈
就是最傳統的燈泡。雖然亮，但有 95% 的電能都變成熱散掉，效率極低。壽命短、老化快，而且沒辦法做頻閃控制。
🟡 螢光燈
便宜、面積大、亮度均勻，曾經是大面積打光的選擇。但光譜不均勻，亮度會隨時間悄悄衰減，同樣沒有閃頻控制。
🟣 氙氣燈（Xenon）
能做超高速閃光（一秒 200 次，每次 1–20μs），光色接近自然日光。但供電系統複雜、成本高昂，現在只剩少數高速特殊應用還在用。
🔵 LED ⭐ 現代工業首選
壽命超過 50,000 小時、發熱量低、亮度精確可調、可做頻閃控制，能組成任意形狀，並可選擇單一波長（紅、綠、藍、IR、NUV）。幾乎沒有明顯缺點，是目前的絕對主流。
🎨
光的顏色也是工具
同色光讓背景消失，互補色光讓邊緣更銳利
▼

你有沒有注意到，在黃色路燈下，鮮豔的顏色看起來都變得暗淡？這是因為特定波長的光打在相同顏色的表面上，那個顏色的對比會被壓低、幾乎消失。反過來，用互補色的光，就能讓那個顏色的區域顯得更深更黑，邊緣更銳利。

實際案例：用紅色 LED 拍一個紅色物件上的黑色印刷文字，紅色背景幾乎消失，黑色文字反而跳出來更清楚。工業上常用紅、綠、藍、白、紅外（IR）和近紫外（NUV），各自在不同材質下有最佳效果——但沒有萬用答案，實際上還是要動手試。

⚡
頻閃控制——讓光源在關鍵瞬間爆發
快門開啟的那一瞬，光源閃出平時數倍的亮度，凍結高速物件
▼

拍高速移動的物件時，曝光時間必須極短，否則影像會模糊。但曝光時間短，進光量就少，影像會太暗。頻閃光源解決了這個矛盾——在相機快門開啟的那一微秒，以平時數倍的強度猛烈閃光，讓影像既夠亮又不模糊。

這樣做還有一個好處：LED 光源平時處於低功率待機，只在必要時全力爆發，實際壽命比連續照明長好幾倍。搭配 Global Shutter 相機使用效果最佳。

影像處理

影像處理是系統的「大腦」，將原始像素資料轉化為有意義的特徵資訊。包含強度轉換、空間濾波、頻率濾波、以及影像分割四大主題。

STEP 01 強度轉換與直方圖

強度轉換（Intensity Transformation）是最基本的影像前處理步驟——對每個像素獨立套用一個數學函數，改變它的亮暗程度，進而提升影像的可視性與對比度。輸入影像的每個像素值 r 都經過函數 S = T(r) 轉換成輸出值 s。

METHOD 門檻化（Thresholding）分割

門檻化是最簡單的分割方式：設定一個亮度門檻值，高於門檻的像素設為白（物體），低於門檻的設為黑（背景）。輸出是二值化影像。

全域門檻化（固定閾值）

整張影像用同一個門檻值。Otsu 演算法能自動計算讓類內變異最小的最佳門檻，是最常用的全域門檻化方法。

自適應門檻化

每個像素根據其局部鄰域計算門檻值，適合光照不均勻的影像——不同區域明暗不同也能正確分割。

METHOD 門檻化（Thresholding）分割

門檻化是最簡單直觀的分割方式：根據像素強度設定閾值 T，高於門檻設為白（前景），低於門檻設為黑（背景）。

整體門檻化
用單一固定門檻值處理整張影像。適合光照均勻、背景單純的場景。
局部門檻化
根據影像局部區域的強度變化動態調整閾值。適合光照不均勻或背景複雜的影像。
動態門檻化
與影像空間座標相關，適合光照不均或背景複雜的影像。

OTSU Otsu 最佳化整體門檻法

目標：自動找出使「類別間變異數」最大化的門檻值 k*。不需要人工選門檻，演算法自動計算出最佳分割點。

// Otsu 演算法步驟
01 計算影像直方圖，正規化為各灰階強度的機率分布 Pᵢ
02 計算群組 C₁（強度 ≤ k）的累積機率 ω(k) 和平均強度 m(k)
03 計算全影像平均強度 m_G，作為基準
04 對每個 k 計算類別間變異數 σ²_B(k)
05 選使 σ²_B(k) 最大的 k 即為最佳門檻 k*
// 互動：拖拉門檻，看類別間變異數
T=0T=127T=255
✅ Otsu vs 人工選值：人工難以從直方圖精確判斷最佳門檻，尤其光照不均時。Otsu 透過最大化類別間變異數自動找到 k*，分割效果顯著優於基本方法，適用於多數場景。評估指標 η = σ²_B / σ²_G 越接近 1 表示分離效果越好。
METHOD 區域為基礎的影像分割

利用像素或子區域的相似性，從種子點開始進行聚合或拆分合併，達成更精細的分割效果。

1. 區域成長法（Region Growing）

從一個或多個種子點出發，根據預先定義的相似性（強度、色度、紋理），不斷將相鄰相似的像素聚合，直到無法加入新的像素為止。

// 演算法步驟
1. 找出影像中與述詞 Q 成立的連通區域
2. 對這些連通區域進行侵蝕，縮至單一像素種子點
3. 以八連通方式，從種子點向外聚合滿足述詞的像素
4. 輸出不同連通區域的標示，完成分割
述詞判斷：若像素與種子點的強度差小於門檻值，述詞為真（加入區域），否則為假（停止）。
2. 區域分裂與合併（Split & Merge）

給定一個述詞 Q，判斷整張影像任一區域是否滿足條件。若述詞為假，將該區域分裂成四個象限；若鄰近區域都滿足述詞，則合併。

// 四叉樹（Quadtree）結構
· 用四叉樹結構表示整個分裂過程
· 每個節點代表一個區域
· 節點不滿足述詞 → 分裂成 4 子節點
· 利用樹狀結構管理區域層級與合併操作
述詞範例：σ > a 且 0 < μ < b，標示為真（TRUE）區域編為1，否則為0。
// 互動示意：點擊種子點，觀察區域成長

// 連通性
